# SmartSplit – Android Expense Splitting App

SmartSplit is an Android application that helps users easily split group expenses and determine who owes whom.
The app automatically calculates balances and minimizes the number of transactions between participants.

---

## Features

• Create group expense splits
• Automatically calculate **who pays whom**
• View split history
• Edit existing splits
• Delete splits
• Share split results
• Dark mode support
• Local data storage using SQLite database

---

## Technologies Used

• Java
• Android Studio
• SQLite Database
• XML UI Design
• RecyclerView and CardView

---

## How the App Works

1. Enter the trip or expense name.
2. Add participants and the amount each person paid.
3. SmartSplit calculates the total expense and each person's share.
4. The app determines the minimum number of transactions required to settle balances.

---

## Installation

1. Download the APK file from this repository.
2. Install it on an Android device.
3. Open the **SmartSplit** app and start managing group expenses.
